Conway's Game of Life

Conway's Game of Life is a cellular automaton devised by the British mathematician John Horton Conway in 1970.

The game is represented by a grid of cells, each of which can be either "alive" or "dead". The grid is given an initial state and updates at a fixed rate. The next generation of the grid is determined by assigning new values to each cell based on the states of each cell's surrounding cells (its "neighbors").

  • Any live cell with fewer than two live neighbors dies, as if by underpopulation.
  • Any live cell with two or three live neighbors lives on to the next generation.
  • Any live cell with more than three live neighbors dies, as if by overpopulation.
  • Any dead cell with exactly three live neighbors becomes a live cell, as if by reproduction.

In this implementation, you can set up the initial grid, start and stop the progression of time at any point, and interact with the grid while it's running.
